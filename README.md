# aquapure-landing-page
🌊 Premium spring water brand landing page. Built with HTML, Tailwind CSS, and vanilla JS. Features full-screen video hero, WhatsApp integration, glass-morphism UI, and fully responsive design for B2B/B2C water delivery business.

A stunning, fully responsive landing page for a premium spring water brand. Built with modern web technologies to showcase products, drive conversions, and provide seamless WhatsApp integration for customer inquiries.

## ✨ Features

- ✅ **Full-Screen Video Hero** - YouTube background with no controls or branding
- ✅ **Glass-Morphism UI** - Premium modern design language
- ✅ **Complete WhatsApp Integration** - All CTAs link directly to WhatsApp
- ✅ **Responsive Design** - Mobile-first, works on all screen sizes
- ✅ **Product Showcase** - 500ml, 1L, 2L, and 5L product display
- ✅ **Video Testimonials** - Embedded YouTube reviews with no controls
- ✅ **B2B Section** - Corporate and event solutions
- ✅ **Floating WhatsApp Button** - Always accessible with pulse animation
- ✅ **Scroll Animations** - Intersection Observer reveal effects
- ✅ **Smooth Navigation** - Sticky header with mobile hamburger menu
- ✅ **Trust Badges** - ISO certification and quality assurances

## 🛠️ Technologies

| Technology | Purpose |
